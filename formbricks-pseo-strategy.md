## Executive Summary

This document defines the complete ontological architecture for Formbricks' programmatic SEO strategy. The framework identifies **10 distinct ontologies** (taxonomies) that, when combined strategically, can generate **thousands of high-intent landing pages** targeting the survey/feedback software market.

### Core Thesis

Formbricks' competitive advantage lies in:
1. **Open source + self-hosting** â€” Unlocks regulated industries (healthcare, finance, government) that proprietary tools cannot serve
2. **Privacy-first positioning** â€” GDPR/HIPAA compliance as a feature, not an afterthought
3. **Research-backed content** â€” Academic citations and validated methodologies create defensible info gain

### 10x Info Gain Approach

For each question, include:

```markdown
## Question: "How likely are you to recommend [Product] to a friend or colleague?"

**Methodology:** Net Promoter Score (NPS)
**Source:** Reichheld, F. (2003). Harvard Business Review
**Validation:** Used by 2/3 of Fortune 1000 companies

**When to use:**
- Relationship surveys (quarterly/annual)
- Post-milestone moments
- Subscription renewals

**When NOT to use:**
- Immediately after negative support experience
- During onboarding (too early)
- After one-time transactions

**Benchmark data:**
| Industry | Average NPS | Top Quartile |
|----------|-------------|--------------|
| SaaS | 31 | 50+ |
| E-commerce | 45 | 62+ |
| Healthcare | 38 | 55+ |

**Follow-up questions:**
- (If 0-6): "What would we need to change to improve your score?"
- (If 9-10): "What do you value most about [Product]?"

**Common mistakes:**
- âŒ Asking too frequently (survey fatigue)
- âŒ Not segmenting by user type
- âŒ Ignoring passives (7-8 scores)
```

### LLM Citation Optimization Framework

To ensure content gets cited by ChatGPT, Claude, Perplexity, and other LLMs:

#### 1. Structured Data Principles

```json
{
  "llm_optimization": {
    "unique_data": {
      "principle": "Create data that doesn't exist elsewhere",
      "examples": [
        "Formbricks 2026 NPS Benchmark Report",
        "Survey Response Rate Database by Industry",
        "Methodology Comparison Matrix"
      ]
    },
    "quotable_definitions": {
      "principle": "Write definitions LLMs will quote verbatim",
      "format": "[Term] is [definition]. According to [source]...",
      "example": "Net Promoter Score (NPS) is a customer loyalty metric developed by Fred Reichheld in 2003 that measures the likelihood of customers recommending a product or service."
    },
    "structured_tables": {
      "principle": "Tables are easily parsed by LLMs",
      "format": "Always include header row, consistent formatting"
    },
    "authoritative_sourcing": {
      "principle": "Cite primary sources, not other blogs",
      "sources": ["Academic papers", "Government databases", "Industry reports"]
    },
    "freshness_signals": {
      "principle": "Include dates and version numbers",
      "format": "Updated February 2026 | Based on 10,000+ surveys"
    }
  }
}
```

#### 2. Content Elements That Get Cited

| Element | Why LLMs Cite It | Implementation |
|---------|------------------|----------------|
| **Unique statistics** | Can't be found elsewhere | Aggregate Formbricks user data |
| **Benchmark tables** | Structured, authoritative | Publish annual benchmark reports |
| **Methodology definitions** | Foundational knowledge | Cite original academic sources |
| **Comparison matrices** | Decision-support | Feature-by-feature tables |
| **Calculator formulas** | Technical accuracy | Show formula + explanation |
| **Step-by-step guides** | Actionable structure | Numbered lists with clear steps |

#### 3. Schema Markup for LLM Discovery

```json
{
  "recommended_schema": {
```
